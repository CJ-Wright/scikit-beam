nsls2.constants.XrayLibWrap_Energy¶

class nsls2.constants.XrayLibWrap_Energy(element, info_type, incident_energy)[source]¶

This is an interface to wrap xraylib to perform calculation on fluorescence cross section, or other incident energy related quantity.

Parameters:

element : int

atomic number

info_type : {‘cs’}

option to calculate physics quantities which depend on incident energy. Valid values are

cs:cross section, unit in cm2/g

incident_energy : float

incident energy for fluorescence in KeV

Examples

Cross section of zinc with an incident X-ray at 12 KeV

>>> x = XrayLibWrap_Energy(30, 'cs', 12)

Compute the cross sec of the Kα1 line.

>>> x['Ka1'] # cross section for Ka1, unit in cm2/g
34.44424057006836
