nsls2.constants.Element¶

class nsls2.constants.Element(element)[source]¶

Object to return all the elemental information related to fluorescence

Parameters:

element : str or int

Element symbol or element atomic Z

Examples

Create an Element object

>>> e = Element('Zn') # or e = Element(30), 30 is atomic number

Get the emmission energy for the Kα1 line.

>>> e.emission_line['Ka1'] #
8.638900756835938

Cross section for emission line Kα1 with 10 keV incident energy

>>> e.cs(10)['Ka1']
54.756561279296875

fluorescence yield for K shell

>>> e.fluor_yield['K']
0.46936899423599243

atomic mass

>>> e.mass
65.37

density

>>> e.density
7.14

Find all emission lines within with in the range [9.5, 10.5] keV with an incident energy of 12 KeV.

>>> e.find(10, 0.5, 12)
{'kb1': 9.571999549865723}

List all of the known emission lines
