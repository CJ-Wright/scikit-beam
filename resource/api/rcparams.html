rcparams Module¶

This module is for dealing with keeping track of package-wide defaults

class nsls2.rcparams.RCParamDict[source]¶

Bases: _abcoll.MutableMapping

A class to make dealing with storing RC params easier. RC params is a hold- over from the UNIX days where configuration files are ‘rc’ files. See http://en.wikipedia.org/wiki/Configuration_file

Examples

Getting and setting data by path is possible

>>> tt = RCParamDict()
>>> tt['name'] = 'test'
>>> tt['nested.a'] = 2
